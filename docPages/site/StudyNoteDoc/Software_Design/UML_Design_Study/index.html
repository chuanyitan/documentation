UML usage study

This page will use the plantUML as design tool to study the UML study. The plantUML documentation is platUML

UML图分为结构图，行为图和交互图: * 结构图分为类图、轮廓图、组件图、组合结构图、对象图、部署图、包图。 * 行为图又分活动图、用例图、状态机图和交互图。 * 交互图又分为序列图、时序图、通讯图、交互概览图。
